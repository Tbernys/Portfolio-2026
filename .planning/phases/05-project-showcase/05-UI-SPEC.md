| Token | Value | Usage |

| xs | 4px | Overlay text gap between title and role |
| sm | 8px | Minimum grid gap (D-04 lower bound) |
| md | 16px | Overlay horizontal padding, lightbox meta internal spacing |
| lg | 24px | Lightbox meta padding below player |
| xl | 32px | Section title bottom margin, lightbox close button offset |
| 2xl | 48px | Section top padding |
| 3xl | 64px | Section bottom padding |

Exceptions:
- Grid gap: 10px (within D-04 range 8-12px, not a strict token multiple but closest even value)
- Lightbox close button: 40px diameter (touch target minimum 44px met by including padding)

## Component Inventory

### 1. Section Container (`#projects`)

- Overrides existing placeholder `min-height: 100vh` and `display: flex`
- Sets `display: block`, `padding: 48px 0 64px`
- Background: #050508 (inherited from existing rule)
- Contains section title + projects grid

### 2. Section Title (`.section-title`)

- Text: "Projets" (D-07)
- Padding: 0 32px 32px
- Color: var(--color-accent) / #D4C5B2
- Font: Inter, 300 weight, uppercase, letter-spacing 0.2em
- Size: clamp(1.25rem, 3vw, 2rem)

### 3. Projects Grid (`.projects-grid`)

- CSS Grid: `grid-template-columns: repeat(3, 1fr)`
- Gap: 10px (D-04)
- Padding: 0 (D-05: edge-to-edge)
- Full width, no max-width constraint

### 4. Project Card (`.project-card`)

- `aspect-ratio: 16 / 9` (D-03)
- `border-radius: 0` (D-06)
- `overflow: hidden`
- `position: relative`
- `cursor: pointer`
- Background: #0a0a0d (fallback)
- Contains: thumbnail img, preview iframe (on hover), overlay

### 5. Thumbnail Image (`.project-thumb`)

- `position: absolute; inset: 0; object-fit: cover`
- `transition: opacity 0.3s ease` (fades when preview iframe appears)
- Source: Vimeo oEmbed API at page load

### 6. Preview Iframe (`.project-preview`)

- Injected into card DOM on mouseenter (desktop) or first tap (mobile)
- `position: absolute; inset: 0; border: none`
- `pointer-events: none` (no user interaction with preview iframe)
- Removed from DOM on mouseleave
- Injection delay: 150ms debounce after mouseenter (prevents flicker on fast cursor pass)

### 7. Card Overlay (`.project-overlay`)

- Position: absolute bottom of card
- Height: auto (content-driven), padding 12px 16px 10px
- Glassmorphism: see color table above
- Border-top only: 1px solid rgba(255, 255, 255, 0.08)
- Default state: `transform: translateY(100%); opacity: 0`
- Active state: `transform: translateY(0); opacity: 1`
- Transition: `transform 0.25s ease-out, opacity 0.25s ease-out`
- Contains: project title (white, semibold) + role (accent, regular)
- Triggered by: `:hover` (desktop) or `.touch-revealed` class (mobile)

### 8. Lightbox (`.lightbox`)

- `position: fixed; inset: 0; z-index: 500`
- Default: `hidden` attribute + `opacity: 0; pointer-events: none`
- Open: remove `hidden`, add `.is-open` on next rAF frame
- `.is-open`: `opacity: 1; pointer-events: auto`
- Transition: `opacity 0.3s ease`
- Close transition: remove `.is-open`, set `hidden` after 350ms
- `aria-modal="true"`, `role="dialog"`
- `document.body.style.overflow = 'hidden'` while open

### 9. Lightbox Backdrop (`.lightbox-backdrop`)

- `position: absolute; inset: 0; z-index: 0`
- Glassmorphism: see color table
- Clickable: closes lightbox (D-12)

### 10. Lightbox Inner (`.lightbox-inner`)

- `position: relative; z-index: 1`
- `width: min(90vw, 1200px)`
- `max-height: 90vh; overflow-y: auto`
- Centered via parent flexbox

### 11. Lightbox Player Wrap (`.lightbox-player-wrap`)

- `width: 100%; aspect-ratio: 16 / 9`
- @vimeo/player SDK injects iframe here
- `responsive: true` in player options

### 12. Lightbox Metadata (`.lightbox-meta`)

- Below player, padding: 24px 0 16px
- Flex column, gap: 4px
- Title: white (#F5F0EB), semibold (600), clamp(1rem, 2.5vw, 1.5rem)
- Role: white (#F5F0EB), regular (400), 14px
- Client: accent (#D4C5B2), regular (400), 14px

### 13. Lightbox Close Button (`.lightbox-close`)

- `position: absolute; top: 24px; right: 24px; z-index: 2`
- Circle: 40px diameter, `border-radius: 50%`
- Glassmorphism: matches existing back-to-top button pattern
- Content: multiplication sign (x) at 1.25rem
- Hover: background brightens to rgba(255, 255, 255, 0.14)
- Transition: background 0.2s

## Interaction States

### Desktop Hover Flow

1. Mouse enters card -> 150ms debounce timer starts
2. Timer fires -> muted autoplay Vimeo iframe injected into card
3. Glassmorphism overlay slides up from bottom (0.25s ease-out)
4. Mouse leaves card -> iframe removed from DOM, overlay slides back down

### Desktop Click Flow

1. Card clicked -> hover preview iframe removed first (prevents audio bleed)
2. Lightbox opens: `hidden` removed, `.is-open` added on next rAF
3. @vimeo/player instance created in lightbox player wrap
4. Metadata populated below player
5. Body scroll locked

### Lightbox Close Flow (3 triggers)

1. Close button (X) click
2. Backdrop click (D-12)
3. Escape key press (a11y — Claude's discretion)
4. All three call same `closeLightbox()`:
   - `player.pause()` then `player.destroy()`
   - Player wrap innerHTML cleared
   - `.is-open` removed, `hidden` set after 350ms
   - Body scroll restored

### Mobile Touch Flow (D-15)

1. First tap on card -> `preventDefault()`, add `.touch-revealed` class
2. Any other revealed cards lose `.touch-revealed`
3. Overlay slides up showing title + role
4. Second tap on same card -> `openLightbox()` called
5. Tap outside any card -> `.touch-revealed` removed from all cards

## Accessibility Contract

| Concern | Implementation |
|---------|---------------|
| Lightbox focus trap | On open: focus moves to close button. On close: focus returns to triggering card |
| Escape key | Closes lightbox (a11y best practice, exercising Claude's discretion per D-12) |
| aria-modal | `aria-modal="true"` on lightbox container |
| role="dialog" | On lightbox container |
| aria-label | "Projets" on section, "Fermer" on close button |
| Reduced motion | `@media (prefers-reduced-motion: reduce)`: disable overlay slide transition, disable hover preview iframe injection, lightbox appears instantly |
| Keyboard nav | Cards are `<button>` or have `tabindex="0"` + Enter/Space triggers lightbox |

## Responsive Behavior

| Breakpoint | Grid | Card size (approx) | Overlay text |
|------------|------|--------------------|----|
| >= 1440px | 3 col, 10px gap, edge-to-edge | ~477px wide | 15px title, 13px role |
| 768-1439px | 3 col, 10px gap, edge-to-edge | ~249px wide | clamp mid-range |
| 375-767px | 3 col, 10px gap, edge-to-edge (D-01) | ~118px wide | 10px title, 9px role (clamp floor) |
| < 375px | 3 col, 10px gap | ~108px wide | clamp floor values |

Per D-01: grid stays 3 columns on ALL viewports. No reflow to 1 or 2 columns. The dense/brute aesthetic is intentional.

Lightbox on mobile: `width: min(90vw, 1200px)` ensures player fills most of the screen. Metadata wraps naturally.
