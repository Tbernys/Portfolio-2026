  <action>
Create `index.html` at the project root with the following structure, all in a single file per TECH-01:

**Head section (in this order):**
1. `<meta charset="UTF-8">`, `<meta name="viewport" ...>`
2. `<title>Tom Bernys | Portfolio</title>`
3. `<script type="importmap">` block FIRST before any module scripts. Map:
   - `"three"` -> `"https://cdn.jsdelivr.net/npm/three@0.175.0/build/three.module.js"`
   - `"three/addons/"` -> `"https://cdn.jsdelivr.net/npm/three@0.175.0/examples/jsm/"`
4. Google Fonts `<link>` tags: preconnect to fonts.googleapis.com and fonts.gstatic.com, then load Comfortaa (weight 700), Inter (variable 100-900), JetBrains Mono (400, 700) with `display=swap`
5. Inline SVG favicon as data URI: 100x100 viewBox, rounded rect (#F5F0EB fill, rx=20), "TB" text centered (font-size 52, weight 700, #000000 fill). URL-encode # as %23.
6. SEO meta description in French: "Tom Bernys, Monteur Video & Motion Designer freelance. Realisations pour des marques exigeantes -- direction artistique, post-production et motion design." (use proper accented characters)
7. Open Graph tags: og:type=website, og:title, og:description, og:image=og-image.jpg (relative path -- no hardcoded domain per user decision)
8. Twitter Card tags: twitter:card=summary_large_image, twitter:title, twitter:description, twitter:image

**Style section (inline `<style>`):**
- Reset: `*, *::before, *::after { box-sizing: border-box; margin: 0; padding: 0; }`
- Body: `background: #000000; color: #F5F0EB; font-family: 'Inter', sans-serif; overflow-x: hidden;`
- Canvas: `position: fixed; top: 0; left: 0; width: 100%; height: 100%; z-index: 0;`
- Main: `position: relative; z-index: 1; pointer-events: none;`
- Interactive elements re-enable: `main a, main button, main form, main input, main textarea { pointer-events: auto; }`
- `.sr-only` class: standard visually-hidden pattern (position absolute, 1px width/height, clip, clip-path inset 50%, overflow hidden, whitespace nowrap)
- Hero section: minimum height 100vh, display flex, align/justify center (centers the visible subtitle)
- Section styling: each section gets min-height to create scroll space for future phases
- Typography: h1 uses Comfortaa, body text uses Inter, set base font sizes
- Color variables as CSS custom properties on :root: `--color-bg: #000000; --color-text: #F5F0EB; --color-accent: #D4C5B2;`

**Body section:**
- `<canvas id="canvas"></canvas>` (the WebGL target)
- `<main>` with `lang="fr"` attribute (anticipate translation per user decision -- use data-lang="fr" on sections)
- Section order per locked decision: Hero -> Clients -> Projects -> Contact
- Hero section: `<section id="hero">` with `<h1>Tom Bernys</h1>` (visible, styled with Comfortaa), `<p class="subtitle">Monteur Video & Motion Designer</p>` (visible, styled with Inter in accent color)
- Clients section: `<section id="clients">` with `<h2 class="sr-only">Clients</h2>` (placeholder for Phase 6)
- Projects section: `<section id="projects">` with `<h2 class="sr-only">Projets</h2>` (placeholder for Phase 5)
- Contact section: `<section id="contact">` with `<h2 class="sr-only">Contact</h2>` (placeholder for Phase 6)
- Use proper French characters with HTML entities where needed (& -> &amp;)

**Script section (inline `<script type="module">`):**
- Import Three.js: `import * as THREE from 'three';`
- Get canvas element, create WebGLRenderer with antialias true
- Set pixel ratio: `Math.min(window.devicePixelRatio, 2)`
- Set renderer size to window dimensions
- Create Scene, PerspectiveCamera (FOV 40, near 0.1, far 100, position z=5)
- Test geometry: BoxGeometry(1, 1, 1) with MeshStandardMaterial color 0xF5F0EB
- Add AmbientLight(0xffffff, 0.5) and DirectionalLight(0xffffff, 1.5) positioned at (5, 5, 5)
- Animate loop: rotate cube on x and y axes (0.005 each frame), render
- Resize handler: update camera aspect ratio, projection matrix, renderer size
- Add a console.log confirming successful initialization: `console.log('[Phase 1] Three.js r' + THREE.REVISION + ' loaded via import map');`

**Important details:**
- `<html lang="fr">` per French language decision
- Do NOT implement any i18n system (deferred idea)
- Use the warm color palette throughout: #000000 bg, #F5F0EB text, #D4C5B2 accent
- The subtitle "Monteur Video & Motion Designer" should be VISIBLE (not sr-only) -- it is the hero tagline. Use accent color (#D4C5B2) and Inter font.
- The h1 "Tom Bernys" should also be visible in Phase 1 as styled text. In Phase 2+ the 3D version will replace it visually, but the HTML h1 remains for accessibility.
