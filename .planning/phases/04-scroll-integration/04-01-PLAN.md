**Track A — CSS-driven (canvas parallax + fade, CTA fade):**
```javascript
const heroTl = gsap.timeline({
  scrollTrigger: {
    trigger: '#hero',
    start: 'top top',
    end: 'bottom top',
    scrub: true,
  }
});

// Hold zone: ~8% of scroll = brief pause
heroTl.to({}, { duration: 0.08 });

// Canvas parallax: translateY -50vh (0.5x scroll speed)
heroTl.to('#canvas', { y: '-50vh', ease: 'none', duration: 0.92 }, 0.08);

// Canvas fade to invisible
heroTl.to('#canvas', { opacity: 0, ease: 'power2.in', duration: 0.92 }, 0.08);

// CTA button fade out by 30% of scroll
heroTl.to('#cta-btn', { opacity: 0, pointerEvents: 'none', ease: 'power1.out', duration: 0.22 }, 0.08);
```

**Track B — Three.js mutations via onUpdate:**
```javascript
ScrollTrigger.create({
  trigger: '#hero',
  start: 'top top',
  end: 'bottom top',
  scrub: true,
  onUpdate: (self) => {
    const progress = self.progress;

    // Glass text Z recession: 3 -> -2 (5 units back), gated on entrance complete
    if (glassGroup && entranceComplete) {
      glassGroup.position.z = 3 - progress * 5;
    }

    // Tile spread uniform
    if (tilesMesh) {
      tilesMesh.material.uniforms.uScrollProgress.value = progress;
    }

    // RAF optimization: stop when canvas invisible
    if (progress > 0.95 && rafId !== null) {
      stopRAFLoop();
    } else if (progress <= 0.95 && rafId === null && !document.hidden) {
      startRAFLoop();
    }
  }
});

// Future phases: call ScrollTrigger.refresh() after adding section content
```
